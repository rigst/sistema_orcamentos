Manual do sistema

Um guia rápido para trabalhar com o StölbenOrçamentos no dia a dia.

1. Cadastros

Comece pelos clientes, depois ajuste categorias, itens do catálogo e dados da empresa.

2. Orçamentos

Crie o orçamento, revise os dados gerais e depois monte os itens com base no catálogo.

3. Status

Use os status para marcar o momento do orçamento: rascunho, enviado, aprovado, rejeitado ou cancelado.

4. Relatórios

Depois de revisar o orçamento, gere PDF ou Excel na tela de relatórios do próprio orçamento.

Fluxo básico

  • Cadastre o cliente com os dados principais.
  • Organize o catálogo por categorias e itens.
  • Confira os dados da empresa para sair corretamente em PDF e Excel.
  • Crie o orçamento, inclua os itens, ajuste totais e revise o status.
  • Use a central de relatórios para exportar ou compartilhar.

Dicas rápidas

  • Os campos monetários aceitam formato brasileiro, como R$ 15.000,00.
  • Clientes, categorias, itens e empresas são inativados em vez de apagados.
  • Ao sair com alterações pendentes, o sistema avisa para evitar perda de dados.
  • Na edição do orçamento, use filtros e ordenação para localizar itens rapidamente.

Fluxo dos status do orçamento

Criar

O orçamento nasce em rascunho. Esse é o momento de montar itens, ajustar valores, revisar descrição e conferir totais.

Enviar

Use quando o orçamento já estiver pronto para apresentação ao cliente. Ele passa a indicar que saiu da montagem interna e foi encaminhado.

Aprovar

Use quando o cliente confirmar a proposta ou quando o processo interno registrar aceite formal do orçamento.

Rejeitar

Use quando o cliente recusar a proposta ou quando o orçamento não seguir adiante comercialmente.

Cancelar

Use quando o orçamento perder a validade operacional, for substituído por outra versão ou precisar sair do fluxo sem ser aprovado nem rejeitado. O cancelamento serve mais para encerrar o documento no histórico.
